Object storage service

Spolehlivé úložiště pro soubory, které nezdržuje.

Ukládejte soubory, hlídejte kvótu, filtrujte metadata a pracujte s přehledným rozhraním postaveným nad FastAPI, SQLAlchemy a Pydantic validací.

Perzistence SQLite + ORM
Validace Pydantic v2
Rozhraní Search, sort, quota
